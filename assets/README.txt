Estructura de assets del proyecto TareasRPG
==========================================

Carpetas
--------
- avatars/      -> retratos del jugador
- items/icons/  -> iconos de armas, armaduras, consumibles y accesorios
- ui/           -> marcos, badges, decoraciones y elementos de interfaz
- vfx/          -> partículas, halos, bursts y efectos de animación

Nota importante
---------------
Por ahora las carpetas se mantienen sin binarios para facilitar versionado inicial.
Cada subcarpeta contiene un README.txt con especificaciones completas de estilo,
colores, tamaños, fondos, nomenclatura y checklist de calidad.
