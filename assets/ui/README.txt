Guía de assets - ELEMENTOS UI 2D
================================

Objetivo
--------
Definir elementos de interfaz (badges, marcos, botones decorativos, divisores, paneles) coherentes con el tema medieval.

Lenguaje visual
---------------
- Materiales: madera oscura, cuero, metal envejecido, dorado.
- Ornamentación moderada (filigranas discretas).
- Bordes redondeados suaves para no romper legibilidad.

Paleta principal (alineada al CSS)
----------------------------------
- Primary: #8B4513
- Secondary: #D2691E
- Accent: #FFD700
- Surface: #3E2418
- BG: #2C1810
- Text main: #F4E4C1
- Text secondary: #D2B48C

Tamaños y exportación
---------------------
- Botones: base 9-slice (para escalar sin deformar bordes)
- Marcos/paneles: 9-slice recomendado
- Badges: 64x64 y 128x128
- Formato: PNG/WebP (transparencia)

Reglas de legibilidad
---------------------
- Contraste mínimo AA para texto sobre paneles.
- No usar texturas agresivas bajo texto pequeño.
- Evitar brillos saturados que compitan con datos numéricos.

Nomenclatura
------------
- ui-panel-wood-01.png
- ui-frame-gold-01.png
- ui-badge-rare-01.png
- ui-divider-ornament-01.png
- ui-button-primary-01.png
